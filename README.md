# NexusCRM | Professional Intelligence Hub

A modern, production-ready Customer Relationship Management and Career Intelligence platform built with Next.js, React, Tailwind CSS, and Firebase.

## Features
- **Resume Vault**: Secure storage for multiple CV versions and live professional links.
- **Document Vault**: Enterprise-grade document management with custom categories.
- **Contact Intelligence**: Professional relationship tracking with detailed profile management.
- **Career Timeline**: Chronological visualization of professional growth.
- **Skills Mapping**: Focused core competency management.

## Getting Started

1. **Setup Environment**: Add your Firebase configuration to `.env.local`.
2. **Install Dependencies**: `npm install`
3. **Run Development Server**: `npm run dev`

## How to Push to GitHub

If you are working in Firebase Studio and want to push your project to GitHub:

1. Create a new repository on [GitHub](https://github.com/new).
2. Open the terminal in your editor.
3. Initialize the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of NexusCRM"
   ```
4. Connect to your GitHub repository:
   ```bash
   git remote add origin <YOUR_GITHUB_REPO_URL>
   git branch -M main
   ```
5. Push the code:
   ```bash
   git push -u origin main
   ```

## Deployment
This app is ready for Firebase App Hosting. Simply connect your GitHub repository in the Firebase Console under App Hosting.
